## Course Overview

The Vibe Coding Foundation course teaches students the fundamentals of web development through hands-on projects and collaborative learning.

## Module 1: Getting Started with Web Development

### Week 1: Introduction to HTML
- Understanding the structure of web pages
- Common HTML elements and tags
- Creating your first webpage

### Week 2: Styling with CSS
- CSS selectors and properties
- Box model and layout basics
- Responsive design fundamentals

### Week 3: JavaScript Basics
- Variables and data types
- Functions and control flow
- DOM manipulation

## Module 2: Modern Web Development

### Week 4: Introduction to React
- Components and JSX
- Props and state
- Event handling

### Week 5: Building with Next.js
- File-based routing
- Server and client components
- API routes

### Week 6: Working with APIs
- Fetch and async/await
- Handling API responses
- Error handling best practices

## Module 3: Real-World Skills

### Week 7: Version Control with Git
- Basic Git commands
- Branching and merging
- Collaborating on GitHub

### Week 8: AI-Assisted Development
- Using AI coding assistants
- Prompt engineering basics
- Building AI-powered applications

## Final Project

Students will build a complete web application that demonstrates:
- Clean, semantic HTML
- Responsive CSS styling
- Interactive JavaScript/React functionality
- API integration
- Version control best practices

## Course Schedule

- **Duration:** 8 weeks
- **Format:** 2 sessions per week, 2 hours each
- **Office Hours:** Available by appointment

- Course Discord server for questions and collaboration
- Weekly code reviews
